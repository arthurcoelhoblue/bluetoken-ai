# Leads Quentes — Sua Mina de Ouro

A tela de **Leads Quentes** (Comercial → Leads Quentes) é uma lista priorizada pela IA de contatos que demonstraram alto interesse recentemente.

## Como Funciona

- A IA analisa comportamento e engajamento de todos os contatos
- Os leads são ordenados por prioridade: **P1** (mais urgente) → **P3**
- Podem não ter um deal aberto ainda — são oportunidades a serem trabalhadas

:::warning Rotina Diária
Verifique esta tela **pelo menos duas vezes por dia**. Um lead quente tem alta probabilidade de conversão se contatado rapidamente.
:::

## O Que Fazer

1. Comece pelo topo da lista (P1)
2. Clique no lead para ver o histórico
3. Inicie o contato — ligue, envie WhatsApp ou e-mail
4. Se houver interesse real, crie um Deal no Pipeline
