# Conversas — WhatsApp e BlueChat

O módulo **Conversas** centraliza todas as interações via WhatsApp e BlueChat em uma interface unificada.

## Funcionalidades

- **Caixa de Entrada Unificada** — Responda sem sair do CRM
- **Contexto Automático** — Cada conversa é vinculada ao contato e seus deals
- **Identificação da IA** — Mensagens da Amélia SDR têm etiqueta "IA"

## Takeover — Assumir a Conversa

Quando a Amélia está conversando com um lead:

1. Você verá uma barra no topo do chat com **"Assumir Conversa"**
2. Clique para pausar a IA e assumir o controle manual
3. Use quando o lead está pronto para falar com um humano

:::tip Devolver para a Amélia
Se você assumiu mas o lead ainda não está pronto, clique em **"Devolver para Amélia"** para que ela continue o follow-up automatizado.
:::
