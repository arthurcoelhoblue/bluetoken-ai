# Cadências — Follow-up no Piloto Automático

Uma **Cadência** é uma sequência de contatos para leads que ainda não estão prontos, mas que você não quer esquecer.

## Como Usar

1. Abra um deal
2. Encontre o card **"Cadência"** na Timeline
3. Clique em **"Adicionar à Cadência"**
4. Escolha uma cadência pré-configurada (ex: "Follow-up de 15 dias")

## O Que Acontece

- O sistema envia e-mails e WhatsApps programados automaticamente
- Tarefas de ligação são criadas para você nos dias certos
- **Se o lead responder, a cadência para automaticamente**

:::info Parada Automática
A cadência é pausada assim que o lead responde, evitando mensagens automáticas durante uma conversa ativa.
:::

## Tipos de Ação em uma Cadência

| Tipo | Execução | Seu Papel |
|:--|:--|:--|
| **WhatsApp** | Automática | Revisar se necessário |
| **E-mail** | Automática | Revisar se necessário |
| **Ligação** | Tarefa criada | Você executa no dia agendado |
