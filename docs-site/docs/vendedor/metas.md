# Metas e Comissões

Na tela de **Metas** (Comercial → Metas), acompanhe seu progresso:

## O Que Você Vê

- **Meta do período** (mês/trimestre)
- **Valor alcançado** até o momento
- **Valor restante** para bater a meta
- **Projeção de comissão** baseada no progresso atual

:::tip Acompanhe Sempre
Consulte esta tela semanalmente para ajustar sua estratégia e saber exatamente quanto falta para atingir seus objetivos.
:::
