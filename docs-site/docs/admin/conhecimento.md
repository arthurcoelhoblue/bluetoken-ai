# Base de Conhecimento — Treinando a Amélia

**Como chegar**: Menu → Configuração → **Base de Conhecimento**

## O Que Fazer

Cadastre produtos e serviços com:
- Descrições detalhadas
- Preços e condições
- Benefícios e diferenciais
- FAQ detalhado por produto

A Amélia usará estas informações para responder a leads de forma autônoma.

## Metodologia de Vendas (Novo!)

Na aba **Metodologia**, envie PDFs de livros e materiais de vendas (SPIN Selling, Challenger Sale, GPCT, etc.). A Amélia absorve as técnicas para moldar **como** ela vende — separado da base de produtos que define **o que** ela vende.

- Os chunks comportamentais são indexados com `source_type = "behavioral"`
- A Amélia aplica as técnicas naturalmente, sem citar o livro ao lead
- Toggle ativo/inativo por material, com reindexação individual

:::danger Mantenha Atualizado
Informações desatualizadas levarão a IA a dar respostas **incorretas** aos clientes. Revise a base de conhecimento semanalmente.
:::
