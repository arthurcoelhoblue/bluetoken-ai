# Webhooks e Integrações Externas

**Como chegar**: Menu → Configuração → **Integrações**

## O Que Fazer

- Configure webhooks para enviar/receber dados de sistemas externos
- Integre com plataformas de marketing (SGT)
- Conecte ERPs e outros sistemas

A documentação de cada webhook detalha o payload esperado.
