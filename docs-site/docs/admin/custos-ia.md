# Monitoramento de Custos de IA

**Como chegar**: Menu → Configuração → **Custos IA**

## Análise Disponível

- Gastos detalhados por **função** (ex: `copilot-chat`, `deal-scoring`)
- Gastos por **modelo de IA**
- Aba **Adoção**: quais funcionalidades estão sendo mais usadas

:::info Rate Limiting
O sistema possui proteções automáticas para evitar picos de custo, mas o monitoramento semanal é recomendado para identificar anomalias.
:::
