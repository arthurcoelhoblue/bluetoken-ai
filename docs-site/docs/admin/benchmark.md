# Benchmark de IA

**Como chegar**: Menu → Configuração → **Benchmark IA**

Ferramenta para testar diferentes modelos de IA em cenários reais do negócio. Use para encontrar o equilíbrio ideal entre **custo** e **qualidade** para cada tarefa.

## Modelos Disponíveis para Teste

- Google Gemini (diversas variantes)
- OpenAI GPT (diversas variantes)

:::tip Otimização
Modelos mais baratos podem ser suficientes para tarefas simples como classificação de intenção, enquanto tarefas complexas como geração de propostas podem exigir modelos mais robustos.
:::
