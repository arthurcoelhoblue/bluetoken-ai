# Configurações da Amélia SDR

**Como chegar**: Menu → Configuração → **Configurações IA**

## O Que Configurar

- Em quais situações a Amélia deve responder automaticamente
- Quando deve escalar para um humano
- Tom de voz utilizado nas respostas
- Frameworks de qualificação (BANT, SPIN, etc.)

:::warning Impacto Direto
Estas configurações afetam diretamente como a Amélia interage com leads. Teste sempre em cenários controlados antes de ativar para produção.
:::
