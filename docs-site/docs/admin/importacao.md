# Importação de Dados

**Como chegar**: Menu → Configuração → **Importação**

## Passo a Passo

1. Prepare um arquivo **CSV** com os dados
2. Acesse a tela de Importação
3. Faça upload do arquivo
4. **Mapeie as colunas** do CSV para os campos do CRM
5. Revise o preview e confirme

:::tip Validação
O sistema valida os dados antes de importar. Erros são listados para correção antes de confirmar.
:::
