# Relatórios e Analytics Executivo

Enquanto o Cockpit mostra o **"o quê"**, o Analytics Executivo mostra o **"porquê"**.

**Como chegar**: Menu → Comercial → Relatórios → Aba "Executivo"

## Insights Preditivos

| Análise | Descrição |
|:--|:--|
| **Previsão de Receita** | Projeção baseada em pipeline, taxas históricas e sazonalidade |
| **Análise de Perda** | Motivos comuns de perda, padrões de objeção, perfis que desistem |
| **Insights de ICP** | Características dos clientes que mais compram |
| **Relatório Semanal** | Resumo automático todo domingo com vitórias, perdas e alertas |

:::info Precisão da Previsão
A precisão da previsão de receita aumenta significativamente após 3 meses de dados históricos.
:::
