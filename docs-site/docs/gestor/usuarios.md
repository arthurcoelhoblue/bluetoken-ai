# Gestão de Usuários e Perfis de Acesso

**Como chegar**: Menu → Configuração → **Usuários**

## Ações

- Convidar novos usuários
- Desativar contas
- Atribuir perfis de acesso (Roles)

:::tip Menor Privilégio
Atribua sempre o perfil com o **menor nível de permissão necessário** para a função (Princípio do Menor Privilégio).
:::
