# Gestão de Funis (Pipelines)

Desenhe o processo de vendas da empresa.

**Como chegar**: Menu → Configuração → **Funis**

## Ações Principais

- **Criar/Editar Etapas** — Nome, cor e **SLA** (tempo máximo em dias)
- **Múltiplos Funis** — Funis diferentes para processos distintos (ex: "Inbound" vs "Enterprise")

:::warning SLA
Etapas com SLA estourado são destacadas em **vermelho** no pipeline do vendedor. Configure SLAs realistas para manter o funil saudável.
:::
