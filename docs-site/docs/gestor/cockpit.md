# Cockpit — O Pulso das Vendas

Central de comando para a operação de vendas.

**Como chegar**: Menu → Comercial → **Cockpit**

## Análises Chave

- **Visão do Funil** — Valor (R$) e quantidade de deals em cada etapa
- **Performance por Vendedor** — Deals ganhos, pipeline gerado, atividades realizadas
- **Velocidade do Funil** — Tempo médio por etapa (identifica gargalos)
- **Taxa de Conversão** — Eficiência do processo etapa por etapa
