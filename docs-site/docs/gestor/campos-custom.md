# Campos Customizados

Adapte o CRM às necessidades de dados da empresa.

**Como chegar**: Menu → Configuração → **Campos**

## O Que Fazer

- Crie campos adicionais para **Deals**, **Contatos** ou **Organizações**
- Defina o tipo: texto, número, data, lista de seleção
- Marque como obrigatório quando necessário
