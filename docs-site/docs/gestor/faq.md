# FAQ do Gestor

### O custo da IA está subindo. Como controlar?

Use o **Benchmark de IA** para testar modelos mais baratos. Verifique em **Custos IA** se alguma função tem uso anormalmente alto (pode indicar erro ou loop).

### A equipe reclama que a IA dá sugestões ruins. O que fazer?

1. Verifique se a **Base de Conhecimento** está completa e atualizada
2. Use o **Benchmark de IA** para comparar modelos
3. Considere refinar os prompts nas configurações

### Como garantir segurança com múltiplos perfis?

Revise periodicamente os perfis na tela de **Usuários**. Aplique sempre o Princípio do Menor Privilégio.
