# FAQ do CSM

### Um cliente novo aparece com Health Score 50. Isso é ruim?

Não. **50 é a pontuação inicial padrão**. A nota real será calculada na primeira rodada diária da IA, após coletar os primeiros dados de uso e engajamento.

### O Risco de Churn subiu muito, mas o cliente parece feliz. Devo confiar na IA?

Sim. A IA detecta padrões sutis que não são óbvios, como diminuição gradual no uso de uma feature chave. Use o alerta como motivo para iniciar uma conversa, mesmo que seja um check-in amigável.

### Como melhorar o Health Score de um cliente?

Foque nos fatores do cálculo:
- Incentive o uso do produto
- Resolva incidências rapidamente
- Mantenha frequência de contato saudável
- Entregue serviço que mereça boa nota de NPS

### Posso usar o Copilot para preparar reuniões de CS?

Sim! Pergunte: *"Resuma a saúde do cliente X e sugira uma pauta para minha próxima reunião."*
