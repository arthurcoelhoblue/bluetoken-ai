# Predição de Churn — Antecipando o Risco

Enquanto o Health Score mede a saúde **atual**, a Predição de Churn olha para o **futuro**. Diariamente, um modelo de IA calcula a probabilidade percentual de cada cliente cancelar nos próximos **90 dias**.

:::tip Proatividade
Esta ferramenta permite focar seus esforços nos clientes que mais precisam de você, **antes** que eles peçam para cancelar.
:::

## Como Agir

| Nível de Risco | Probabilidade | Ação Recomendada |
|:--|:--|:--|
| **Alto** | > 50% | Ação imediata: agende ligação ou reunião |
| **Moderado** | 25-50% | Inicie playbook de engajamento, envie check-in |
| **Baixo** | < 25% | Monitoramento normal |
