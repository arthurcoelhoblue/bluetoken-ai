# Manual de Sucesso do Cliente

**Da retenção ao encantamento: gestão de clientes com IA**

No Amélia CRM, a Amélia trabalha 24/7 para monitorar a saúde dos seus clientes, prever riscos, automatizar tarefas e fornecer insights para conversas estratégicas.

:::info Seu objetivo
Construir relacionamentos sólidos. O objetivo da Amélia é garantir que você saiba exatamente com quem falar e sobre o que falar, antes que um problema se torne crítico.
:::

## Rotina Diária do CSM

### Manhã (Primeira Hora)
1. Abra o **Dashboard CS** para visão geral da carteira
2. Verifique **Clientes em Risco** (alta predição de churn)
3. Analise o **Briefing Diário** gerado pela Amélia

### Durante o Dia
4. Gerencie **Incidências** abertas
5. Execute tarefas dos **Playbooks**
6. Use o **Copilot** para preparar reuniões

### Final do Dia
7. Analise respostas de **Pesquisas (NPS/CSAT)**
8. Atualize notas nos perfis dos clientes
9. Planeje ações para o dia seguinte
