# Pesquisas — A Voz do Cliente

O Amélia CRM automatiza o envio de dois tipos de pesquisa:

## NPS (Net Promoter Score)

- **Pergunta**: "De 0 a 10, o quanto você nos recomendaria?"
- **Mede**: Lealdade geral
- **Frequência**: Enviado periodicamente de forma automática

## CSAT (Customer Satisfaction)

- **Pergunta**: "De 1 a 5, quão satisfeito você ficou com este atendimento?"
- **Mede**: Satisfação com interação específica
- **Quando usar**: Após resolução de uma incidência

## Visualização

Todas as respostas (notas e comentários) ficam disponíveis na tela de **Pesquisas**.
