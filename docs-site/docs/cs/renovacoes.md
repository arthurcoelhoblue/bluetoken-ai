# Renovações — Gestão de Contratos

Conforme a data de renovação se aproxima, o sistema alerta o CSM automaticamente.

## Ferramentas para Renovação

- **Alertas automáticos** com antecedência configurável
- **Histórico de saúde** do cliente como base para a negociação
- **Identificação de oportunidades** de upsell baseada no uso
- **Tracking de status** da renovação no sistema

:::tip Preparação
Use o histórico de Health Score e NPS como argumentos durante a negociação de renovação. Clientes saudáveis são candidatos naturais para upsell.
:::
