# Dashboard CS — Visão Panorâmica da Carteira

Sua tela principal: um resumo executivo da saúde de todo o seu portfólio de clientes.

## Métricas Chave

| Métrica | O Que Mede |
|:--|:--|
| **MRR Total** | Receita mensal recorrente — o valor que você protege |
| **Clientes em Risco** | Quantidade com churn > 50% (calculado pela IA) |
| **Health Score Médio** | Média da pontuação de saúde de todos os clientes |
| **NPS Médio** | Média de Net Promoter Score — lealdade geral da base |

:::tip Tendências
Use este dashboard para identificar tendências. Se o número de clientes em risco sobe semana após semana, uma ação mais ampla pode ser necessária.
:::
